## 启动指南

### 前置准备

#### 1. 获取 API Key

访问 DeepSeek 平台获取 API Key：
- DeepSeek: https://platform.deepseek.com/

#### 2. 配置环境变量

创建 `.env` 文件或修改 `backend/config.json`：

```env
API_KEY=your_deepseek_api_key
BASE_URL=https://api.deepseek.com/v1
MODEL=deepseek-v4-pro
EMBEDDING_MODEL=deepseek-chat
```

### 启动服务

#### 1. 启动 MCP 服务（工具服务）

```bash
cd backend
pip install -r requirements.txt
python mcp_module/start.py
```

MCP 服务器运行在: `http://localhost:8080/mcp`

#### 2. 启动后端服务

```bash
cd backend
python app.py
```

后端运行在: `http://localhost:5000`

#### 3. 启动前端服务

```bash
cd frontend
npm install
npm run dev
```

前端运行在: `http://localhost:5173`

#### 4. 访问应用

浏览器打开: `http://localhost:5173`

## 功能说明

- **顶部**: 显示"智能客服"标题
- **中间**: 聊天内容区域
  - 左侧: 客服回复（白色气泡）
  - 右侧: 用户消息（紫色渐变气泡）
- **底部**: 输入框和发送按钮
  - 支持回车发送
  - 支持点击按钮发送

## 技术栈

**后端:**
- Flask 3.0.0 - Web 框架
- Flask-CORS 4.0.0 - 跨域支持
- Flask-Limiter 3.5.0 - 限流支持
- OpenAI SDK >= 1.40.0 - AI 对话（兼容 DeepSeek）
- LangChain >= 0.3.0 - Agent、工具框架、ExampleSelector
- langchain-chroma >= 0.1.0 - Chroma 向量存储集成
- langchain-community >= 0.3.0 - 社区工具和嵌入模型
- Chroma >= 0.5.0 - 向量数据库
- MCP - Model Context Protocol

**前端:**
- React 18 - UI 框架
- Vite - 构建工具
- Axios - HTTP 请求

## 工具系统

系统支持以下工具调用：

1. **天气查询** (`get_weather`) - 查询指定城市的实时天气
2. **天气预报** (`get_weather_forecast`) - 查询指定城市未来几天天气预报
3. **表单提交** (`submit_form`) - 提交客户咨询表单

工具调用规则：
- 工具通过 MCP 服务器统一管理和调用
- Agent 自动判断并选择合适的工具

## 自定义配置

### 修改系统提示词

编辑 `backend/modules/prompt/__init__.py` 中的 `CUSTOMER_SERVICE_PROMPT_TEMPLATE` 变量

### 自定义 FewShot 示例

编辑 `backend/modules/prompt/__init__.py` 中的 `DEFAULT_FEW_SHOT_EXAMPLES` 列表

### 添加新工具

1. 在 `mcp_module/tools/` 创建插件文件
2. 使用 `@register_tool` 装饰器注册工具
3. 在 `mcp_module/start.py` 中导入工具模块
4. 重启 MCP 服务器

## 常见问题

**Q: 启动时提示 API Key 错误?**
A: 检查 `.env` 或 `config.json` 文件是否正确配置了 `api_key`

**Q: 向量化失败?**
A: 检查网络连接，确认 API Key 有 embedding 权限

**Q: 如何重置知识库?**
A: 删除 `db/` 目录下的向量数据文件，重启服务即可

**Q: MCP 服务器端口被占用?**
A: 使用 `taskkill /F /PID <PID>` 终止占用进程，或等待端口释放

**Q: 工具调用失败?**
A: 检查 MCP 服务器是否正常运行，确保应用能访问 `http://localhost:8080/mcp`

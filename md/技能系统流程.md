## 一、流程概述

技能系统通过 `SKILL.md` 定义技能，支持语义匹配和脚本执行：

| 步骤 | 功能 | 模块 |
|------|------|------|
| **意图识别** | 识别为 SKILL 类别意图 | `IntentRouter` |
| **技能匹配** | 根据语义匹配技能 | `SkillMatcher` |
| **指令解析** | 解析技能指令 | `SkillExecutor` |
| **脚本执行** | 执行技能脚本 | `SkillRunner` |

## 二、完整流程图

```mermaid
flowchart TB
    subgraph Phase1["阶段1: 意图识别"]
        A[用户查询] --> B[IntentRouter.route]
        B --> C[识别为 SKILL 类别]
        C --> D[Intent: category=SKILL]
    end
    
    subgraph Phase2["阶段2: 技能匹配"]
        D --> E[SkillExecutor]
        E --> F[提取 target 技能名]
        F --> G{有明确 target?}
        G -->|有| H[直接加载技能]
        G -->|无| I[语义匹配技能]
        I --> J[遍历 SKILL.md]
        J --> K[计算相似度]
        K --> L[选择最佳匹配]
        L --> H
    end
    
    subgraph Phase3["阶段3: 指令解析"]
        H --> M[加载 SKILL.md]
        M --> N[解析 instructions]
        N --> O[匹配用户指令]
        O --> P[提取指令参数]
    end
    
    subgraph Phase4["阶段4: 脚本执行"]
        P --> Q[SkillRunner]
        Q --> R[执行脚本/函数]
        R --> S[获取执行结果]
        S --> T[格式化为 ExecutionResult]
    end
    
    subgraph Phase5["阶段5: 结果返回"]
        T --> U[存入 intent_results]
        U --> V[CallModelNode]
        V --> W[生成最终回答]
    end
    
    style A fill:#e1f5fe,color:#01579b
    style D fill:#c8e6c9,color:#1a5e20
    style H fill:#fff3e0,color:#e65100
    style T fill:#f3e5f5,color:#7b1fa2
    style W fill:#bbdefb,color:#0d47a1
```

## 三、技能定义结构

### 3.1 SKILL.md 结构

```markdown
# 技能名称

## 基本信息
- 名称: drawio-skill
- 描述: 使用DrawIO创建各种图表
- 版本: 1.0.0

## 指令列表
### 指令1: 创建流程图
- 触发词: 流程图, 流程, diagram
- 参数: title, nodes, edges
- 示例: 创建一个用户登录流程图

### 指令2: 创建思维导图
- 触发词: 思维导图, mindmap
- 参数: topic, branches
- 示例: 创建一个项目规划思维导图

## 执行脚本
- 路径: scripts/create_diagram.py
- 函数: create_flowchart(title, nodes, edges)
```

### 3.2 技能元数据

```python
SkillMetadata(
    name="drawio-skill",
    description="使用DrawIO创建各种图表",
    version="1.0.0",
    instructions=[
        SkillInstruction(
            trigger=["流程图", "diagram"],
            action="create_flowchart",
            params=["title", "nodes", "edges"],
        ),
    ],
    script_path="scripts/create_diagram.py",
)
```

## 四、技能匹配机制

### 4.1 语义匹配流程

```mermaid
flowchart LR
    subgraph Match["语义匹配流程"]
        A[用户指令] --> B[提取关键词]
        B --> C[遍历技能 trigger]
        C --> D{关键词匹配?}
        D -->|匹配| E[计算相似度]
        D -->|不匹配| F[继续遍历]
        E --> G[选择最高相似度]
        F --> C
        G --> H[返回技能指令]
    end
    
    style Match fill:#c8e6c9,color:#1a5e20
```

### 4.2 匹配优先级

| 优先级 | 匹配方式 | 说明 |
|--------|----------|------|
| **1** | 明确 target | Intent 中指定技能名 |
| **2** | 关键词匹配 | 用户指令包含 trigger 词 |
| **3** | 语义相似度 | 向量相似度匹配 |

## 五、技能执行流程

### 5.1 脚本执行流程

```mermaid
flowchart TB
    subgraph Execute["脚本执行流程"]
        A[技能指令] --> B[解析参数]
        B --> C[加载脚本模块]
        C --> D[调用执行函数]
        D --> E[执行结果]
        E --> F{执行成功?}
        F -->|成功| G[返回结果]
        F -->|失败| H[错误处理]
        H --> I[返回错误信息]
    end
    
    style Execute fill:#fff3e0,color:#e65100
```

### 5.2 参数提取

```python
# 用户输入: "创建一个用户登录流程图"

    "edges": [("开始", "输入账号"), ("输入账号", "验证"), ...],
}
```

## 七、扩展指南

### 7.1 新增技能步骤

1. **创建技能目录**

```
server/skills/my-skill/
├── SKILL.md          # 技能定义
├── scripts/          # 执行脚本
│   └── main.py
└── references/       # 参考文档（可选）
```

2. **编写 SKILL.md**

```markdown
# my-skill

## 基本信息
- 名称: my-skill
- 描述: 我的自定义技能

## 指令列表
### 指令1: 执行某操作
- 触发词: 关键词1, 关键词2
- 参数: param1, param2
```

3. **实现执行脚本**

```python
# scripts/main.py
def execute_action(param1, param2):
    # 执行逻辑
    return {"result": "执行结果"}
```

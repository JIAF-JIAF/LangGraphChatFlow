# 智能客服 Agent - 快速启动指南

## 前置准备

### 1. 获取 API Key

访问阿里云百炼或 OpenAI 平台获取 API Key：
- 阿里云百炼: https://www.aliyun.com/product/dashscope
- OpenAI: https://platform.openai.com/

### 2. 配置环境变量

**使用 config.json 文件**

```json
{
  "api_key": "your_api_key",
  "base_url": "https://dashscope.aliyuncs.com/compatible-mode/v1",
  "model": "qwen-plus",
  "embedding_model": "text-embedding-v3"
}
```

| 配置项 | 说明 |
|--------|------|
| api_key | 阿里云百炼或 OpenAI API 密钥 |
| base_url | API 基础地址 |
| model | 对话模型名称 |
| embedding_model | 向量化模型名称 |

## 启动服务

### 方式一: 完整启动（推荐）

**步骤 1: 启动 MCP 服务器（工具服务）**

打开终端，进入 backend 目录：

```bash
cd backend
python mcp_module/start.py
```

MCP 服务器运行在: `http://localhost:8080/mcp`

**步骤 2: 启动应用服务器**

打开另一个终端，进入 backend 目录：

```bash
cd backend
python app.py
```

后端运行在: `http://localhost:5000`

### 方式二: 使用测试脚本

终端1 启动 MCP 服务:
```bash
cd backend
python mcp_module/start.py
```

终端2 启动应用服务:
```bash
cd backend
python app.py
```

终端3 运行测试:
```bash
cd backend
python test.py
```

## 自定义配置

### 修改知识库

编辑 `knowledge_base/` 目录下的文件，添加业务知识。支持多个知识库：
- `knowledge_base/default/` - 默认知识库
- `knowledge_base/politics/` - 政策文档知识库

### 修改系统提示词

编辑 `modules/prompt/__init__.py` 中的 `CUSTOMER_SERVICE_PROMPT_TEMPLATE` 常量

### 自定义 FewShot 示例

编辑 `modules/prompt/__init__.py` 中的 `DEFAULT_FEW_SHOT_EXAMPLES` 列表：

```python
DEFAULT_FEW_SHOT_EXAMPLES = [
    {"user_query": "你好", "assistant_response": "您好！请问有什么可以帮助您的？"},
    {"user_query": "你们有什么产品?", "assistant_response": "我们提供多种优质产品..."},
]
```

### 添加新工具

1. 在 `mcp_module/tools/` 创建插件文件
2. 使用 `@register_tool` 装饰器注册工具
3. 在 `mcp_module/start.py` 中导入工具模块
4. 重启 MCP 服务器

```python
from mcp_module.tools.registry import register_tool

@register_tool(
    name="my_tool",
    description="我的自定义工具",
    parameters=[
        {"name": "param1", "type": "string", "description": "参数说明", "required": True}
    ],
    return_type="string"
)
def my_tool(param1: str) -> str:
    return "工具执行结果"
```

## 常见问题

**Q: 启动时提示 API Key 错误?**
A: 检查 `.env` 或 `config.json` 文件是否正确配置了 `api_key`

**Q: 向量化失败?**
A: 检查网络连接，确认 API Key 有 embedding 权限

**Q: 如何重置知识库?**
A: 删除 `db/` 目录下的向量数据文件，重启服务即可

**Q: MCP 服务器端口被占用?**
A: 使用 `taskkill /F /PID <PID>` 终止占用进程，或等待端口释放

**Q: 工具调用失败?**
A: 检查 MCP 服务器是否正常运行，确保应用能访问 `http://localhost:8080/mcp`

**Q: StateModifier 导入失败?**
A: StateModifier 在新版 LangGraph 中已移除，代码已兼容处理

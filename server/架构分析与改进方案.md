## 一、当前架构评估

### 1.1 分层设计（评分：⭐⭐⭐⭐ 较好）

```
┌─────────────────────────────────────────────────────────┐
│  表现层 (app.py, api/)                                  │
├─────────────────────────────────────────────────────────┤
│  调度层 (modules/langgraph/agent.py)                    │
├─────────────────────────────────────────────────────────┤
│  执行层 (modules/assistant.py)                          │
├─────────────────────────────────────────────────────────┤
│  业务层 (modules/rag/, modules/skill/, modules/feeling/)│
├─────────────────────────────────────────────────────────┤
│  基础层 (modules/checkpoint/, modules/logger/)          │
└─────────────────────────────────────────────────────────┘
```

#### 优点

- 层次清晰，职责边界明确
- LangGraph 作为调度层，LangChain Agent 作为执行层，符合业界最佳实践
- 依赖注入设计良好（`factory.py`）
- 支持多种存储后端（Memory、Redis）

#### 问题

| 问题 | 位置 | 影响 |
|------|------|------|
| API 目录与 modules 职责重叠 | `api/skill_installer.py` | 违反分层原则 |

| 路由逻辑分散 | `rag/router/`, `skill/matcher/` | 难以统一管理 |

## 二、多意图识别问题分析

### 2.1 问题描述

**用户输入：**
```
先帮我查询一个行测绝杀蒙题技巧，再帮我画一个架构图
```

**期望结果：**
1. 查询知识库，返回蒙题技巧
2. 调用画图技能，生成架构图

**实际结果：**
- 只返回了蒙题技巧
- 未识别到画图意图
- 未执行画图技能

---

### 2.2 当前流程分析

```
START → feeling_detect → router → ┌── retrieve → plan
                                    │
                                    └── plan ←─────┘
                                              │
                                              ▼
                              execute_task → check_task_complete → call_model → END
```

#### 节点职责

| 节点 | 职责 | 输入 | 输出 |
|------|------|------|------|
| feeling_detect | 情绪检测 | query | feeling |
| router | 判断是否需要 RAG | query | need_retrieve |
| retrieve | 执行 RAG 检索 | query | documents |
| plan | 任务规划 | query, documents | subtasks |
| execute_task | 执行任务 | subtask | result |
| call_model | 生成最终回答 | answer | final_answer |

#### 问题根因

```
1. feeling_detect: 检测情绪 ✅
2. router: 判断是否需要 RAG 检索 → 返回 True（因为包含"查询"）✅
3. retrieve: 执行 RAG 检索 → 返回蒙题技巧文档 ✅
4. plan: 任务规划 → 只生成一个任务（基于难度评估）❌
   - task_planner 基于难度评估拆分任务，不是意图识别
   - 难度评估为 2 级（简单推理），只生成 1 个任务
5. execute_task: 执行任务 → 只回答蒙题技巧 ❌
6. call_model: 生成最终回答 → 缺少画图部分 ❌
```

#### 核心问题

1. **没有多意图识别节点** - router 只判断是否需要 RAG，不识别多意图
2. **task_planner 不是意图识别器** - 它基于难度评估拆分任务，不是意图识别
3. **技能执行依赖被动触发** - 技能通过 tool calling 被动触发，缺乏主动意图识别

---

### 2.3 业界最佳实践对比

#### 多意图处理模式

| 模式 | 描述 | 适用场景 | 框架支持 |
|------|------|----------|----------|
| **Orchestrator-Worker** | 中央编排器识别意图，分发给专门 Worker | 多领域任务 | LangGraph, AutoGen |
| **Plan-and-Execute** | 先规划完整计划，再逐步执行 | 复杂多步骤任务 | LangGraph |
| **Intent Decomposition** | 专门的意图分解节点，识别并拆分多意图 | 多意图请求 | 自定义 |
| **Command API** | LangGraph Command API 动态路由 | 运行时决策 | LangGraph 1.0+ |

#### 参考架构（LangGraph 官方推荐）

```python
from langgraph.types import Command
from typing import Literal

def intent_decompose_node(state) -> Command[Literal["rag_worker", "skill_worker", "chat_worker"]]:
    """意图分解节点：识别并拆分多意图"""
    intents = llm_analyze_intents(state["query"])
    
    if len(intents) > 1:
        # 多意图：创建子任务队列
        return Command(
            update={
                "intents": intents,
                "current_intent_idx": 0,
                "multi_intent_mode": True
            },
            goto="intent_router"
        )
    else:
        # 单意图：直接路由
        return Command(goto=get_worker_for_intent(intents[0]))
```

##### 3. 意图识别器实现

```python
# modules/intent/recognizer.py
from typing import List, Dict, Any
from langchain_core.messages import HumanMessage
from dataclasses import dataclass

@dataclass
class Intent:
    """意图对象"""
    type: str           # 意图类型
    content: str        # 意图内容
    order: int          # 执行顺序
    confidence: float   # 置信度

class IntentRecognizer:
    """意图识别器"""
    
    INTENT_TYPES = {
        "rag_query": "知识库查询",
        "skill_execute": "技能执行",
        "general_chat": "普通对话",
    }
    
    def __init__(self, llm_client):
        self.llm = llm_client
    
    def recognize(self, query: str) -> List[Intent]:
        """识别用户意图"""
        prompt = self._build_prompt(query)
        response = self.llm.chat.invoke([HumanMessage(content=prompt)])
        return self._parse_response(response.content)
    
    def _build_prompt(self, query: str) -> str:
        return f"""
你是一个意图识别专家。请分析用户请求，识别所有意图。

用户请求：{query}

意图类型说明：
- rag_query: 知识库查询（查询、搜索、查找、检索）
- skill_execute: 技能执行（画图、绘图、生成图表、架构图、流程图）
- general_chat: 普通对话（闲聊、问候、感谢）

多意图识别规则：
1. 如果用户使用"先...再..."、"然后"、"接着"等连接词，表示多个意图
2. 每个意图应独立执行，按顺序返回结果
3. 技能执行意图应识别具体的技能类型（如画图、生成文档等）

请返回 JSON 格式：
{{
    "is_multi_intent": true/false,
    "intents": [
        {{
            "type": "意图类型",
            "content": "意图具体内容",
            "order": 执行顺序（从1开始）,
            "confidence": 置信度（0-1）
        }}
    ]
}}
"""
    
    def _parse_response(self, response: str) -> List[Intent]:
        """解析 LLM 响应"""
        import json
        try:
            data = json.loads(response)
            return [
                Intent(
                    type=item["type"],
                    content=item["content"],
                    order=item["order"],
                    confidence=item.get("confidence", 1.0)
                )
                for item in data.get("intents", [])
            ]
        except Exception as e:
            # 解析失败，返回默认意图
            return [Intent(type="general_chat", content=response, order=1, confidence=0.5)]
```

##### 4. 意图分解器实现

```python
# modules/intent/decomposer.py
from typing import List, Dict, Any
from .recognizer import Intent

class IntentDecomposer:
    """意图分解器：将多意图拆分为子任务"""
    
    def decompose(self, intents: List[Intent]) -> List[Dict[str, Any]]:
        """将意图列表转换为子任务列表"""
        subtasks = []
        
        for intent in intents:
            task = self._intent_to_task(intent)
            subtasks.append(task)
        
        return subtasks
    
    def _intent_to_task(self, intent: Intent) -> Dict[str, Any]:
        """将单个意图转换为任务"""
        TASK_MAPPING = {
            "rag_query": {
                "task_type": "rag",
                "handler": "retrieve",
                "description": f"查询知识库：{intent.content}"
            },
            "skill_execute": {
                "task_type": "skill",
                "handler": "skill_executor",
                "description": f"执行技能：{intent.content}"
            },
            "general_chat": {
                "task_type": "chat",
                "handler": "call_model",
                "description": f"对话：{intent.content}"
            }
        }
        
        task_template = TASK_MAPPING.get(intent.type, TASK_MAPPING["general_chat"])
        
        return {
            "task_id": f"task_{intent.order}",
            "task_type": task_template["task_type"],
            "handler": task_template["handler"],
            "task_description": task_template["description"],
            "intent": intent,
            "status": "pending",
            "result": ""
        }
```

```python
# application/orchestrator/orchestrator.py
from typing import List, Dict, Any
from modules.intent import IntentRecognizer, IntentDecomposer

class Orchestrator:
    """编排器：统一管理意图识别和任务分发"""
    
    def __init__(self, llm_client):
        self.intent_recognizer = IntentRecognizer(llm_client)
        self.intent_decomposer = IntentDecomposer()
        self.task_dispatcher = TaskDispatcher()
        self.result_aggregator = ResultAggregator()
    
    def process(self, query: str, session_id: str) -> Dict[str, Any]:
        """处理用户请求"""
        # 1. 识别意图
        intents = self.intent_recognizer.recognize(query)
        
        # 2. 分解为子任务
        subtasks = self.intent_decomposer.decompose(intents)
        
        # 3. 分发执行
        results = []
        for task in subtasks:
            result = self.task_dispatcher.dispatch(task, session_id)
            results.append(result)
        
        # 4. 聚合结果
        final_answer = self.result_aggregator.aggregate(results)
        
        return {
            "answer": final_answer,
            "intents": intents,
            "results": results
        }
```

```python
# infrastructure/events/event_bus.py
from typing import Callable, Dict, List
from dataclasses import dataclass

@dataclass
class Event:
    """事件基类"""
    type: str
    data: Dict
    timestamp: float

class EventBus:
    """事件总线"""
    
    def __init__(self):
        self._subscribers: Dict[str, List[Callable]] = {}
    
    def publish(self, event: Event):
        """发布事件"""
        handlers = self._subscribers.get(event.type, [])
        for handler in handlers:
            handler(event)
    
    def subscribe(self, event_type: str, handler: Callable):
        """订阅事件"""
        if event_type not in self._subscribers:
            self._subscribers[event_type] = []
        self._subscribers[event_type].append(handler)

# 使用示例
event_bus = EventBus()

# 订阅意图识别完成事件

def on_intent_recognized(event: Event):
    intents = event.data["intents"]
    # 执行后续处理
    pass

# 发布事件
event_bus.publish(Event(
    type="intent_recognized",
    data={"intents": intents, "session_id": session_id},
    timestamp=time.time()
))
```

```python
    @wraps(func)

        span_id = generate_span_id()

    return wrapper
```

```python
# modules/langgraph/nodes/intent.py

from dataclasses import dataclass

@dataclass
class Intent:
    """意图对象"""
    type: str
    content: str
    order: int
    confidence: float

class IntentNode:
    """意图识别节点"""
    
    def __init__(self, llm_client):
        self.llm = llm_client
    
    def __call__(self, state: Dict[str, Any]) -> Dict[str, Any]:
        """节点入口"""
        query = state["query"]
        intents = self.recognize(query)
        
        is_multi = len(intents) > 1
        
        return {
            "intents": intents,
            "is_multi_intent": is_multi,
            "current_intent_idx": 0
        }
    
    def recognize(self, query: str) -> List[Intent]:
        """识别意图"""
        prompt = f"""
分析用户请求，识别所有意图。

用户请求：{query}

意图类型：
- rag_query: 知识库查询（查询、搜索、查找）
- skill_execute: 技能执行（画图、分析、生成）
- general_chat: 普通对话

返回 JSON：
{{
    "is_multi_intent": true/false,
    "intents": [
        {{"type": "意图类型", "content": "意图内容", "order": 顺序, "confidence": 置信度}}
    ]
}}
"""
        
        response = self.llm.chat.invoke([HumanMessage(content=prompt)])
        return self._parse_response(response.content)
    
    def _parse_response(self, response: str) -> List[Intent]:
        """解析响应"""
        import json
        try:
            data = json.loads(response)
            return [
                Intent(
                    type=item["type"],
                    content=item["content"],
                    order=item["order"],
                    confidence=item.get("confidence", 1.0)
                )
                for item in data.get("intents", [])
            ]
        except:
            return [Intent(type="general_chat", content=response, order=1, confidence=0.5)]
```

```python
# modules/langgraph/agent.py 修改

from langgraph.types import Command
from typing import Literal

class LangGraphAgent:
    
    def _intent_recognize_node(self, state: AgentState) -> AgentState:
        """意图识别节点"""
        query = state["query"]
        intents = self._intent_node.recognize(query)
        
        return {
            "intents": intents,
            "is_multi_intent": len(intents) > 1,
            "current_intent_idx": 0
        }
    
    def _should_decompose(self, state: AgentState) -> Literal["router", "intent_router"]:
        """条件路由：是否需要分解多意图"""
        if state.get("is_multi_intent", False):
            return "intent_router"
        return "router"
    
    def _intent_router_node(self, state: AgentState) -> Command[Literal["retrieve", "skill_executor", "call_model"]]:
        """意图路由节点：根据当前意图路由到对应处理器"""
        intents = state["intents"]
        current_idx = state["current_intent_idx"]
        
        if current_idx >= len(intents):
            # 所有意图已处理完成
            return Command(goto="call_model")
        
        current_intent = intents[current_idx]
        
        # 路由表
        ROUTING_TABLE = {
            "rag_query": "retrieve",
            "skill_execute": "skill_executor",
            "general_chat": "call_model"
        }
        
        target = ROUTING_TABLE.get(current_intent.type, "call_model")
        
        return Command(
            update={
                "current_intent": current_intent,
                "current_intent_idx": current_idx
            },
            goto=target
        )
    
    def _build_graph(self):
        """构建状态图（新增意图识别）"""
        
        self._graph = StateGraph(AgentState)
        
        # 添加节点
        self._graph.add_node("feeling_detect", self._feeling_detect_node)
        self._graph.add_node("intent_recognize", self._intent_recognize_node)  # 新增
        self._graph.add_node("router", self._router_node)
        self._graph.add_node("intent_router", self._intent_router_node)  # 新增
        self._graph.add_node("retrieve", self._retrieve_node)
        self._graph.add_node("skill_executor", self._skill_executor_node)  # 新增
        self._graph.add_node("plan", self._plan_node)
        self._graph.add_node("execute_task", self._execute_task_node)
        self._graph.add_node("check_task_complete", self._check_task_complete_node)
        self._graph.add_node("call_model", self._call_model_node)
        
        # 基础流程
        self._graph.add_edge(START, "feeling_detect")
        self._graph.add_edge("feeling_detect", "intent_recognize")  # 新增
        
        # 意图识别后的条件路由（新增）
        self._graph.add_conditional_edges(
            "intent_recognize",
            self._should_decompose,
            {"router": "router", "intent_router": "intent_router"}
        )
        
        # 单意图路径
        self._graph.add_conditional_edges(
            "router",
            self._should_retrieve,
            {"retrieve": "retrieve", "plan": "plan"}
        )
        
        # 多意图路径（新增）
        self._graph.add_edge("intent_router", "retrieve")  # 或 skill_executor
        
        # 其他边...
        self._graph.add_edge("retrieve", "plan")
        self._graph.add_edge("plan", "execute_task")
        # ...
        
        self._graph = self._graph.compile(checkpointer=self._checkpointer)
```

```python
class SkillMatcher:
    
    def match_by_intent(self, intent: Intent) -> Optional[Dict[str, Any]]:
        """根据意图主动匹配技能"""
        if intent.type != "skill_execute":
            return None
        
        # 提取技能关键词
        keywords = self._extract_skill_keywords(intent.content)
        
        # 语义匹配
        skill = self.match_by_semantic(keywords)
        
        if skill:
            log(f"意图匹配技能成功: {skill['name']}", "Skill.Matcher")
            return skill
        
        # 关键词兜底
        return self.match_by_keywords(intent.content)
    
    def _extract_skill_keywords(self, content: str) -> str:
        """提取技能关键词"""
        # 技能关键词映射
        SKILL_KEYWORDS = {
            "drawio-skill": ["画图", "绘图", "架构图", "流程图", "时序图", "类图"],
            "data-analysis": ["分析", "数据分析", "统计", "报表"],
            "trip-plan": ["旅行", "旅游", "行程", "攻略"],
        }
        
        for skill_name, keywords in SKILL_KEYWORDS.items():
            if any(kw in content for kw in keywords):
                return " ".join(keywords)
        
        return content
```

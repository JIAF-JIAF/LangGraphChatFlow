# 旅行规划技能

## 基本信息

- **名称**: trip-plan
- **标题**: 旅行规划技能
- **版本**: 1.0.0
- **作者**: AI Team

## 技能描述

帮助用户制定旅行计划，包括目的地推荐、行程安排、交通预订、住宿推荐等一站式服务。

## 触发条件

### 关键词触发
- 旅行
- 旅游
- 行程
- 攻略
- 计划

### 难度等级
4

## 执行流程

### 步骤一：需求收集
- **目标**: 了解用户旅行需求
- **操作**: 询问关键信息（目的地、时间、预算、人数、偏好）
- **输入**: 用户初始查询
- **输出**: 结构化需求

### 步骤二：目的地推荐（如未指定）
- **目标**: 根据偏好推荐合适目的地
- **操作**: 调用推荐工具或知识库检索
- **条件**: 用户未指定目的地
- **工具**: [get_destination_recommend]

### 步骤三：交通查询
- **目标**: 查询往返交通方式
- **操作**: 查询机票、火车票、自驾路线等
- **输入**: 出发地、目的地、时间
- **输出**: 交通方案列表
- **工具**: [search_flights, search_trains]

### 步骤四：住宿推荐
- **目标**: 根据需求推荐住宿
- **操作**: 查询酒店、民宿等
- **输入**: 目的地、预算、入住时间
- **输出**: 住宿方案列表
- **工具**: [search_hotels]

### 步骤五：行程规划
- **目标**: 制定详细每日行程
- **操作**: 结合景点、美食、活动规划路线
- **输入**: 目的地、天数、偏好
- **输出**: 每日行程安排
- **工具**: [get_attractions, get_restaurants]

### 步骤六：预订协助
- **目标**: 帮助用户完成预订
- **操作**: 调用预订工具或提供预订链接
- **条件**: 用户确认预订
- **工具**: [book_flight, book_hotel]

### 步骤七：生成计划文档
- **目标**: 输出完整旅行计划
- **操作**: 整合所有信息生成文档
- **输出**: 旅行计划文档
- **工具**: [export_plan]

## 工具依赖

| 工具名称 | 说明 | 必填 |
|----------|------|------|
| get_destination_recommend | 目的地推荐 | 否 |
| search_flights | 机票查询 | 是 |
| search_trains | 火车票查询 | 否 |
| search_hotels | 酒店查询 | 是 |
| get_attractions | 景点查询 | 是 |
| get_restaurants | 餐厅推荐 | 否 |
| book_flight | 机票预订 | 否 |
| book_hotel | 酒店预订 | 否 |
| export_plan | 导出计划文档 | 是 |

## 专业知识

### 旅行类型
- **休闲度假**: 放松身心，享受美景
- **文化探索**: 历史古迹、博物馆
- **美食之旅**: 特色美食体验
- **户外探险**: 登山、徒步、露营

### 预算等级
- **经济型**: 性价比优先
- **舒适型**: 品质与价格平衡
- **豪华型**: 高端体验

### 参考文档
- [中国领事服务网](http://cs.mfa.gov.cn/)
- [国家移民管理局](https://s.nia.gov.cn/)

## 示例

**输入**: 帮我规划一个3天的杭州旅行

**输出**:
```markdown
# 旅行计划

## 基本信息
- **目的地**: 杭州
- **日期**: 2024-03-15 - 2024-03-17
- **天数**: 3天
- **人数**: 1人
- **预算**: 舒适型

## 交通安排
- 去程: G7501 上海虹桥→杭州东 08:00-08:45
- 返程: G7502 杭州东→上海虹桥 18:00-18:45

## 住宿安排
- 酒店: 杭州西湖国宾馆
- 地址: 西湖区杨公堤18号
- 价格: ¥800/晚

## 每日行程

### Day 1: 西湖风光
- 09:00 - 西湖断桥残雪
- 11:30 - 楼外楼午餐
- 14:00 - 雷峰塔景区
- 17:00 - 西湖音乐喷泉

### Day 2: 灵隐古刹
- 08:30 - 灵隐寺
- 11:00 - 龙井村品茶
- 14:00 - 九溪十八涧
- 17:30 - 知味观晚餐

### Day 3: 文化体验
- 09:00 - 浙江省博物馆
- 11:30 - 河坊街小吃
- 14:00 - 西溪湿地
- 16:00 - 返程

## 注意事项
1. 西湖景区周末人流量大，建议早出发
2. 灵隐寺门票需提前预约
3. 春季杭州多雨，带好雨具
```
